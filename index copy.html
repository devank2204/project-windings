Frequently Asked Questions

Everything you need to know about Wingdings and our translator

What are Wingdings?

Wingdings is a decorative font created by Microsoft that replaces standard letters with symbols, arrows, and pictographs. Originally released in 1990, it has become iconic in internet culture.

What is Gaster mode?

Gaster mode converts text using the mysterious character W.D. Gaster's language from Undertale, which primarily uses uppercase Wingdings symbols for cryptic dialogue.

Can I download my converted text?

Yes! You can export your Wingdings text as PNG or SVG files for use in designs, presentations, or any creative project.

Does this work on mobile?

Absolutely! Our translator is fully responsive and optimized for mobile devices with touch-friendly interactions and smooth animations.

FAQs

Is this real Wingdings?

We use Unicode lookalikes that resemble Wingdings. Actual Wingdings is a symbol font, not a Unicode block.

What is Gaster?

Gaster refers to Wingdings-styled text from the game Undertale. This tool approximates that look.
